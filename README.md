# Rick and Morty API
Atividade de api, do tema Rick and Morty, desenvolvida com HTML, CSS e JavaScript

# Funcionalidades

Buscar personagens por id;
Buscar personagens por nome;
Avançar personagem;
Voltar personagem;
Tradução das informações dos personagens exceto o nome;
Tratamento de erros com try/catch

# Foi utilizado:

HTML;
CSS;
JavaScript;
Rick and Morty API;

# API utilizado:
https://rickandmortyapi.com/
